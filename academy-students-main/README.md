# Academy Gatekeeper

Simple authentication service with signup and login functionality.

## Features

- User signup and login
- Password encryption
- JSON file storage

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

## API Endpoints

- `GET /` - Welcome message
- `POST /signup` - Register new user
- `POST /login` - User login
